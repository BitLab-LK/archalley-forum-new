# Advertisement Banner Requirements

## Banner Sizes and Locations

### 1. Leaderboard Banner (970x180px)
- Location: After hero section
- Purpose: High-visibility top placement
- Example: Conference announcements, major promotions

### 2. Horizontal Banner (680x180px)  
- Location: Between content sections (Blog->Projects, Articles->News)
- Purpose: Content break advertisements
- Example: Services, portfolio promotions

### 3. Square Banner (350x350px)
- Location: Sidebar (sticky position)
- Purpose: Persistent sidebar advertising
- Example: Software tools, courses, materials

## Implementation Features

✅ Responsive design - adapts to screen sizes
✅ Click tracking - logs ad interactions  
✅ Random rotation - different ads on each visit
✅ Hover effects - interactive user experience
✅ Fallback images - graceful error handling
✅ External links - opens in new tab
